R Coding Basics

Posted on 2020-07-26 | In note
Reading time ≈ 20 mins

The summary of the basic operations in R which benefit my PhD study.

Packages

Install:

  • install.packages("xxx")

Load in R:

  • require(xxx): gives a confirmation message
  • library(xxx): no message


Graphics

Simple plot:

  • plot(x), plot(x, y): automatically choose the best plot type

  • plot(fcn, xmin, xmax): plot the function fcn

Bar chart:

  • table(aFrame): to summarize the class for bar charts
  • barplot()

Histogram:

  • hist(): create a simple histogram

Scatter plot:

  • plot(x, y)

Subplot:

  1. par(mfrow=c(#row, #col)): set the graphical parameter
  2. plot separately
  3. par(mfrow=c(1, 1)): restore graphic parameter


Data Analysis

summary():

  • summary(iris$Species) # Categorical variable
  • summary(iris$Sepal.Length) # Quantitative variable
  • summary(iris) # Entire data frame

iris is a dataset in library(datasets)


describe() from psych package:

It gets more details. A complement for histogram or box plot.

It only takes quantitative variables.

output information: vars, n, mean, sd, median, trimmed, mad, min, max, range, skew, kurtosis, se


Selecting cases using aFrame[condition1 & condition2 ..]

select by category or value or both


Data Formats

In contrast to other programming languages like C and java in R, the variables are not declared as some data type. The variables are assigned with R-objects and the data type of the R-object becomes the data type of the variable. R-objects include vector, matrix/array, data frame, list, and factor.

Vector

  • all same data type
  • The simplest of these objects is the vector object and there are six data types of these atomic vectors, also termed as six classes of vectors.
    • Logical
    • Numeric
    • Integer
    • Complex
    • Character
    • Raw
  • define: v <- c(1, 2, 3, 4, 5)

Matrix/array

  • two dimensions (array 3+D)

  • same data class

  • columns not named
  • define:
    • matrix: m <- matrix(c(T, T, F, F, T, F), nrow = 2)
    • array: a <- array(c(1:24), c(4, 3, 2)) # Give data, then dimensions (rows, columns, tables)

Data frame

  • two dimensions and can have vectors of multiple types
  • all same length
  • closest R analogue to spreadsheet
  • define: df <- as.data.frame(cbind(vNumeric, vCharacter, vLogical)) where vNumeric, vCharacter and vLogical are vectors of numeric, character and logical.

List

  • most flexible
  • ordered collection of elements
  • any class, length, or structure
  • can include lists
  • define: list2 <- list(vNumeric, vCharacter, vLogical, list1)

Factor

It stores the vector along with the distinct values of the elements in the vector as labels. The labels are always character irrespective of whether it is numeric or character or Boolean etc. in the input vector. They are useful in statistical modeling.

Coercing types

Coerce numeric to integer:

  • coerce3 <- as.integer(5)

Coerce character to numeric:

  • coerce5 <- as.numeric(c("1", "2", "3"))

Coerce matrix to data frame:

  • coerce7 <- as.data.frame(matrix(1:9, nrow=3))


Entering Data

  • x:y: ascending or descending order

  • seq(x, y, by = aStride)

  • c(x1, x2, ...): concatenate as a vector

  • scan(): input from console window
  • rep(): replicate existing data


Importing Data

rio (R Import Output) package combines all of R’s import functions into one simple utility.

  • import("aPath"): import .csv, .txt, .xlsx, etc. file

R’s built-in functions:

  • read.table()
  • read.csv()


RStudio Hotkey

  • Clear console: ctrl + L
  • Run line by line: ctrl + enter
  • Fast dash <-: alt + -
