Grammar of Graphics

  1. Data

    • A tidy format

  2. Mapping

    • Aesthetic mapping: Link variables in data to graphical properties in the geometry.

    • Facet mapping: Link variables in the data to panels in the facet layout.

  3. Statistics: data -> displayed values

    • Implicit in many plot-types

  4. Scales: variable ranges -> property ranges

    • Categories -> Colour

    • Numbers -> Position

  5. Geometries
    • Bar / line / point … a high degree determinator of your plot type
  6. Facets
    • Deﬁne the number of panels with equal logic and split data among them.
    • Allows you to look at small subsets of your data in separate plots.
  7. Coordinates
    • Physical position …
  8. Theme
has sensible defaults

Statistics

The stat can be overwritten. If we have precomputed count we don’t want any additional computations to perform and we use the identity stat to leave the data alone.

Values calculated by the stat is available with the after_stat() function inside aes(). You can do all sorts of computations inside that. Every geom has a stat. This is why new data (count) can appear when using geom_bar().

  • after_stat(): An aesthetic expression using variables calculated by the stat.

Facets

Features:

  • Split data into multiple panels

  • Each panel is a representation of the same underlying logic (avoid over plotting)

  • Should not be used to combine multiple separate plots
  • ggplot2 provide two facets for splitting data by categories
    • facet_wrap: Wrap a 1d ribbon of panels into 2d
    • facet_grid: It is most useful when you have two discrete variables, and all combinations of the variables exist in the data.


Exercises:

1) Subplots have independent scales:

scales argument: Be fixed (“fixed”, the default), free (“free”), or free in one dimension (“free_x”, “free_y”)?

2) Subplots have proportional spaces:

space argument: If “fixed”, the default, all panels have the same size. If “free_y” their height will be proportional to the length of the y scale; if “free_x” their width will be proportional to the length of the x scale; or if “free” both height and width will vary.

Coordinates

The difference between scales and coordinates:

  • Scales: apply in the beginning (select data -> further processing)
  • Coordinates: apply at the end (further processing -> create windows for display)

Themes
